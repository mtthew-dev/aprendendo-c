# 🧠 O que é a Linguagem C?

A linguagem C é uma das linguagens mais importantes da história da computação.

Criada por Dennis Ritchie nos laboratórios Bell, ela influenciou diretamente diversas linguagens modernas como:

* C++
* Java
* Python
* JavaScript
* C#
* Rust
* Go

C é conhecida por:

✔ Alta performance
✔ Controle de memória
✔ Eficiência
✔ Proximidade com o hardware
✔ Forte base acadêmica e profissional
